IPython Notebook¶


(Jupyter)¶

IPython Notebook¶

  • 类似 Mathematica Notebook
  • 可以展示文本、公式、代码、计算结果、图形表格、音频、视频
  • 支持 Markdown, LaTeX Math
  • 语法高亮和命令补全
  • 储存在 .ipynb 文件中
  • 可以导出为 pdf 或 html

优点¶

  • 同时展示代码与结果
  • 方便传播交流，可重复研究

安装¶

  • 推荐使用 Anaconda
  • 推荐以非 root 用户安装

使用¶

  1. 终端输入 ipython notebook 或 jupyter-notebook
  2. 在浏览器中访问地址 http://localhost:8888

使用服务器上的notebook¶

  • 在服务器上打开 notebook
    • ipython notebook --no-browser --port=8888
  • 将服务器 127.0.0.1:8888 转发到本地 2222 端口（这些端口都可任意选择）
    • ssh -L 2222:127.0.0.1:8888 user@host_server
  • 在本地访问
    • http://127.0.0.1:2222/
  • 非常重要
    • 为数据安全考虑，采用了端口转发的 notebook 设置密码与 https 加密连接是必须的
    • 如果没有设置密码，同服务器上的人可以通过 notebook 任意访问你的文件！
    • 设置方法参考这里

图与子图¶

  • figure：图，可以看做画布
    • 用 figure 函数生成 Figure 对象
  • axes：子图，一个图中可以有多个子图
    • 用 subplot 生成 Axes 对象
    • 用 subplots 一次性生成多个子图
  • gcf/gca 获得当前图/子图
  • scf/sca 设定当前图/子图
  • clf/cla 清空当前图/子图

Numpy中多个条件筛选¶

numpy bool 数组支持用 ~ & | 做逻辑运算，注意这里不能用not/and/or。

data = rand(10)
x, y, z = rand(10), rand(10), rand(10)
a, b, c = 0.3, 0.7, 0.5

idx = (x > a) | (y < b) & (z > c)
data_select = data[idx]

不少人遇到过这个问题，所以附在这里。
